# Lumora 产品战略报告

> 日期：2026-07-09 | 数据来源：项目文档 + 公开行业知识 | 网络检索受限，部分数据标注"假设"

---

## 1. 市场洞察摘要

**市场背景**：AI 图像生成工具（Stable Diffusion、Midjourney、ComfyUI）用户量爆发式增长。用户单设备积累数百至数千张生成图片，现有文件管理器无法按 prompt/seed/模型等 AI 原生维度检索。（来源：Civitai 2025 年报称月活超 2000 万；Hugging Face 2025 AI 图像模型下载量超 5 亿次——**待验证，网络受限**）

**核心痛点**：
1. **检索低效**：Windows 资源管理器/Mac Finder 无法搜索 prompt、seed 等元数据
2. **隐私焦虑**：云端图库（如 Eagle Cloud）上传 AI 生成图片存在版权和隐私风险
3. **变体管理缺失**：同 prompt 不同 seed 的对比散落在文件夹各处
4. **语义搜索空白**：无法用"月光下的森林"找到对应图片

**增长机会**：本地优先 + AI 原生元数据 + 语义搜索 = 蓝海细分。全球 AI 图像生成用户预估 5000 万+（**假设，基于 Civitai/Stability AI 公开数据推算，待验证**），其中需要专业管理工具的比例约 10-20%。

## 3. 差异化策略与护城河

**核心差异化**：
1. **AI 原生元数据体系**：prompt/seed/model/采样器字段级搜索 + 变体自动分组——竞品均不支持
2. **本地语义搜索**：Ollama + sqlite-vec 实现零云端依赖的自然语言检索——技术门槛高
3. **极简美学**："古卷·灯火"设计语言在工具类软件中独树一帜——情感溢价

**可持续护城河**：
- **技术护城河**：Rust + SQLite + sqlite-vec 的本地向量搜索栈，竞品难以短期复制
- **社区护城河**：MIT 开源 + AI 创作者社区口碑
- **数据护城河**：用户积累的标签/评分/变体关系形成迁移成本

## 4. 产品架构与核心功能清单

**当前架构**（v0.5.1）：Tauri 2 + React 19 + Rust + SQLite + Ollama

**核心功能**：
- 图片导入（拖拽/文件夹，自动提取 PNG 元数据）
- 多维搜索（FTS5 全文 + 字段范围 + 语义 + 视觉相似）
- 标签/评分/收藏系统
- AI 分析（llava 自动标注 + 调色板提取）
- 变体追踪（同 prompt 自动分组）
- 回收站 + 批量操作
- 数据库备份/恢复
- 暗色主题 + 国际化

**待建设**（对标用户需求）：
- ❌ 移动端（iOS/Android）
- ❌ Web 版
- ❌ 实时协同
- ❌ 离线/在线无缝切换

## 5. 分阶段实施路线图

### MVP（当前 → v1.0）— 桌面端深耕
**目标**：成为 AI 创作者首选的本地图库
- 补齐：批量嵌入生成、导出模板增强、性能优化（万张级）
- 时间：2-3 个月
- 指标：GitHub Star 1000+，活跃用户 500+

### V1（v1.0 → v2.0）— 移动端扩展
**目标**：跨设备图片浏览（单向同步）
- 新增：iOS/Android 只读浏览端（Tauri Mobile 或 React Native）
- 同步方案：局域网 WiFi 直传（非云同步，保持隐私）
- 时间：6-9 个月
- 指标：移动端 DAU 200+

### V2（v2.0 → v3.0）— 协同与 Web
**目标**：团队协作场景
- 新增：Web 端只读分享、端到端加密同步、多人标注
- 时间：12-18 个月
- 指标：付费用户转化率 5%

**注意**：用户要求"躲避蓝海赛道"，但当前产品本身已在蓝海（AI 图片管理细分）。建议策略：**在蓝海中建立壁垒，而非回避**。移动端和协同是红海方向（Google Photos、iCloud），应谨慎投入。

## 6. 关键风险与应对建议

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| Ollama 生态萎缩 | 低 | 高 | 抽象 AI 层，支持多后端（llama.cpp、ONNX） |
| Eagle 推出 AI 搜索 | 中 | 高 | 深耕 prompt/seed 原生字段，Eagle 架构难以复制 |
| 移动端投入过大 | 中 | 中 | 先做只读浏览端，不做编辑 |
| 协同功能偏离核心 | 高 | 中 | 坚持本地优先，协同仅限局域网场景 |
| 用户增长瓶颈 | 中 | 高 | 加入 ComfyUI/A1111 插件生态，降低发现门槛 |

**关键假设（待验证）**：
1. AI 图像生成用户中 10-20% 有专业管理需求——需用户调研
2. 本地语义搜索体验优于云端——需 A/B 测试
3. 用户愿意为"变体追踪"功能迁移工具——需访谈验证
