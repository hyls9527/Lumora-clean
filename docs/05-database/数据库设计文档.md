**版本**: 1.0  
**日期**: 2026-07-09  
**状态**: 初稿

## 1. 数据库概述

### 1.1 数据库选型
- **数据库**: SQLite 3.x
- **存储模式**: WAL (Write-Ahead Logging)
- **字符编码**: UTF-8
- **文件位置**: `data/lingjian.db`

### 1.2 设计原则
- **规范化**: 遵循3NF范式，减少数据冗余
- **完整性**: 使用外键约束保证数据一致性
- **性能**: 合理创建索引，优化查询性能
- **可扩展**: 预留扩展字段，支持未来功能

## 2. 表结构设计

### 2.1 任务表 (tasks)

#### 2.1.1 表定义
```sql
CREATE TABLE tasks (
    id TEXT PRIMARY KEY,
    file_path TEXT NOT NULL,
    file_name TEXT NOT NULL,
    file_size INTEGER NOT NULL,
    file_hash TEXT NOT NULL,
    duration REAL NOT NULL,
    status TEXT NOT NULL DEFAULT 'pending',
    progress REAL DEFAULT 0.0,
    model_size TEXT DEFAULT 'medium',
    language TEXT DEFAULT 'auto',
    beam_size INTEGER DEFAULT 5,
    checkpoint TEXT,
    error_message TEXT,
    gpu_device INTEGER DEFAULT 0,
    priority INTEGER DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    started_at TIMESTAMP,
    completed_at TIMESTAMP,
    
    -- 约束
    CHECK (status IN ('pending', 'transcribing', 'completed', 'failed', 'paused', 'cancelled')),
    CHECK (progress >= 0 AND progress <= 100),
    CHECK (file_size > 0),
    CHECK (duration > 0)
);
```

#### 2.1.2 索引设计
```sql
-- 状态索引（用于查询待处理任务）
CREATE INDEX idx_tasks_status ON tasks(status);

-- 创建时间索引（用于排序）
CREATE INDEX idx_tasks_created_at ON tasks(created_at);

-- 优先级索引（用于优先级队列）
CREATE INDEX idx_tasks_priority ON tasks(priority DESC);

-- 文件哈希索引（用于检测重复文件）
CREATE INDEX idx_tasks_file_hash ON tasks(file_hash);
```

### 2.2 片段表 (segments)

#### 2.2.1 表定义
```sql
CREATE TABLE segments (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    task_id TEXT NOT NULL,
    segment_index INTEGER NOT NULL,
    start_time REAL NOT NULL,
    end_time REAL NOT NULL,
    speaker TEXT,
    speaker_id INTEGER,
    text TEXT NOT NULL,
    confidence REAL,
    words TEXT,
    language TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    -- 外键约束
    FOREIGN KEY (task_id) REFERENCES tasks(id) ON DELETE CASCADE,
    
    -- 约束
    CHECK (start_time >= 0),
    CHECK (end_time > start_time),
    CHECK (confidence IS NULL OR (confidence >= 0 AND confidence <= 1)),
    
    -- 唯一约束
    UNIQUE (task_id, segment_index)
);
```

#### 2.2.2 索引设计
```sql
-- 任务ID索引（用于查询任务的所有片段）
CREATE INDEX idx_segments_task_id ON segments(task_id);

-- 说话人索引（用于按说话人筛选）
CREATE INDEX idx_segments_speaker ON segments(speaker);

-- 时间范围索引（用于时间查询）
CREATE INDEX idx_segments_time_range ON segments(start_time, end_time);
```

### 2.3 标记表 (marks)

#### 2.3.1 表定义
```sql
CREATE TABLE marks (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    segment_id INTEGER NOT NULL,
    tags TEXT,
    note TEXT,
    color TEXT DEFAULT '#1a73e8',
    mark_type TEXT DEFAULT 'custom',
    is_highlighted BOOLEAN DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    -- 外键约束
    FOREIGN KEY (segment_id) REFERENCES segments(id) ON DELETE CASCADE,
    
    -- 约束
    CHECK (mark_type IN ('important', 'follow_up', 'question', 'confirmed', 'invalid', 'custom'))
);
```

#### 2.3.2 索引设计
```sql
-- 片段ID索引（用于查询片段的所有标记）
CREATE INDEX idx_marks_segment_id ON marks(segment_id);

-- 标记类型索引（用于按类型筛选）
CREATE INDEX idx_marks_type ON marks(mark_type);
```

### 2.4 审计日志表 (audit_logs)

#### 2.4.1 表定义
```sql
CREATE TABLE audit_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    action TEXT NOT NULL,
    entity_type TEXT NOT NULL,
    entity_id TEXT NOT NULL,
    details TEXT,
    user_id TEXT,
    ip_address TEXT,
    timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    -- 约束
    CHECK (action IN ('INSERT', 'UPDATE', 'DELETE', 'SELECT', 'EXPORT', 'IMPORT'))
);
```

#### 2.4.2 索引设计
```sql
-- 实体索引（用于查询实体的操作历史）
CREATE INDEX idx_audit_logs_entity ON audit_logs(entity_type, entity_id);

-- 时间戳索引（用于时间范围查询）
CREATE INDEX idx_audit_logs_timestamp ON audit_logs(timestamp);
```
