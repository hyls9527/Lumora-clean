## 1. 变体对比视图 (Variant Comparison View)

### WBS 分解

| WBS ID | 原子任务 | 类型 | 依赖 | 测试目标 |
|--------|---------|------|------|---------|
| V8-001 | 前端 API: `getVariantGroupImages` | API | - | 调用正确 IPC，返回 ImageRecord[] |
| V8-002 | Store: `variantStore` | Store | V8-001 | 状态管理：加载、数据、错误 |
| V8-003 | 组件: `VariantCompareModal` | UI | V8-002 | 渲染 N 张图片，支持切换 |
| V8-004 | 集成: DetailModal 中添加"查看变体"按钮 | 集成 | V8-003 | 按钮显示/隐藏逻辑 |

### 数据流

```
DetailModal (图片有 variant_group_id?)
  → 点击"查看变体"
  → variantStore.getVariants(variantGroupId)
  → API: invoke('get_variant_group_images', { variantGroupId })
  → VariantCompareModal (展示 2-6 张图片并排)
```
