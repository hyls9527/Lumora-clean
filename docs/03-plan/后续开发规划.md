## 4. 开发规范

### 4.1 测试编写规范

```typescript
// 测试文件命名
src/features/xxx/__tests__/XxxPage.test.tsx

// 测试结构
describe('XxxPage', () => {
  it('should render correctly', () => {});
  it('should handle user interaction', () => {});
  it('should display error state', () => {});
});
```

### 4.2 质量门禁

每个 Sprint 结束前必须通过：
```bash
npx tsc --noEmit      # TypeScript 检查
npx vitest run        # 前端测试
cargo test --lib      # Rust 测试
```

### 4.3 提交规范

```
test(xxx): add tests for XxxPage
feat(xxx): add xxx feature
fix(xxx): fix xxx bug
```
