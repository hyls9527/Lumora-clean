# 部署环境架构文档 — Lumora v0.5.1

> 基于 `.github/workflows/` 和 `src-tauri/tauri.conf.json` 实际配置生成。

## 1. CI/CD 流程

### 1.1 持续集成 (ci.yml)

```
触发条件：push to main / PR to main

┌─────────────────────┐  ┌─────────────────────┐
│   Frontend Job      │  │   Rust Job          │
│   (ubuntu-latest)   │  │   (ubuntu-latest)   │
├─────────────────────┤  ├─────────────────────┤
│ 1. npm ci           │  │ 1. rust toolchain   │
│ 2. tsc --noEmit     │  │ 2. cargo cache      │
│ 3. vitest run       │  │ 3. apt deps         │
│ 4. npm run build    │  │ 4. cargo test --lib │
│                     │  │ 5. cargo check      │
└─────────────────────┘  └─────────────────────┘
```

**依赖安装（Ubuntu）：**
- libgtk-3-dev, libwebkit2gtk-4.1-dev, libappindicator3-dev, librsvg2-dev, patchelf

### 1.2 发布构建 (release.yml)

```
触发条件：push tag v*

┌─────────────────────────────────────────────────┐
│              Build Matrix                        │
├─────────────────┬────────────────┬───────────────┤
│  ubuntu-latest  │ windows-latest │  macos-latest │
│  x86_64-linux   │ x86_64-windows │ aarch64-darwin│
├─────────────────┼────────────────┼───────────────┤
│ AppImage + DEB  │ MSI + NSIS EXE │     DMG       │
└────────┬────────┴───────┬────────┴───────┬───────┘
         │                │                │
         └────────┬───────┘                │
                  ▼                        │
         ┌───────────────┐                 │
         │ GitHub Release │◄───────────────┘
         │ (draft)       │
         └───────┬───────┘
                 ▼
         ┌───────────────┐
         │ updater-json  │
         │ 生成 latest   │
         │ .json         │
         └───────────────┘
```

**签名配置：**
- `TAURI_SIGNING_PRIVATE_KEY` — GitHub Secret
- `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` — GitHub Secret
- updater JSON 中 signature 字段当前为空字符串

### 1.3 安全审计 (security-audit.yml)

```
触发条件：每周定时 + 手动

┌─────────────────────┐
│ npm audit           │
│ cargo audit         │
└─────────────────────┘
```

## 2. 构建产物

| 平台 | 格式 | 文件名模式 |
|------|------|-----------|
| Windows | MSI | `lumora_{version}_x64_en-US.msi` |
| Windows | NSIS EXE | `lumora_{version}_x64-setup.exe` |
| Linux | AppImage | `lumora_{version}_amd64.AppImage` |
| Linux | DEB | `lumora_{version}_amd64.deb` |
| macOS | DMG | `lumora_{version}_aarch64.dmg` |

## 3. 自动更新

```
┌──────────┐     ┌──────────────┐     ┌──────────────┐
│ 客户端   │────►│ GitHub       │────►│ latest.json  │
│ 启动     │     │ Releases API │     │ (per-platform)│
└──────────┘     └──────────────┘     └──────────────┘
     │
     ▼
┌──────────────────┐
│ 比较版本号        │
│ 有新版本 → 提示   │
│ 用户确认 → 下载   │
│ 下载完成 → 安装   │
└──────────────────┘
```

**latest.json 结构：**
```json
{
  "version": "0.5.1",
  "notes": "",
  "pub_date": "2026-07-07T00:00:00Z",
  "platforms": {
    "windows-x86_64": { "signature": "", "url": "..." },
    "darwin-aarch64": { "signature": "", "url": "..." },
    "linux-x86_64": { "signature": "", "url": "..." }
  }
}
```

## 4. 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `OLLAMA_HOST` | `http://localhost:11434` | Ollama 服务地址 |

## 5. 本地开发环境

```bash
# 前置依赖
node >= 20
rust stable
Ollama (可选，AI 功能需要)

# 安装
npm ci

# 开发
npm run dev          # 启动 Vite dev server + Tauri

# 测试
npx vitest run       # 前端测试
cd src-tauri && cargo test --lib  # Rust 测试
npx tsc --noEmit     # 类型检查

# 构建
npm run build        # 构建前端
cargo build --release # 构建 Rust 后端
```

## 6. CSP 安全策略

```
default-src 'self'
script-src 'self'
style-src 'self' 'unsafe-inline' https://fonts.googleapis.com
img-src 'self' asset: https://asset.localhost
font-src 'self' https://fonts.gstatic.com
connect-src 'self' http://localhost:* https://localhost:* https://github.com ipc: tauri:
```

**外部连接：**
- `localhost:*` — Ollama API
- `github.com` — 自动更新检查
- `fonts.googleapis.com` / `fonts.gstatic.com` — Google Fonts
