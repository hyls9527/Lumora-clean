## 3. 测试用例设计

### 3.1 批量管理测试用例

#### 3.1.1 TaskRepository测试
```python
# tests/test_task_repository.py
import pytest
from lingjian.batch.repository import TaskRepository
from lingjian.batch.models import Task

class TestTaskRepository:
    """TaskRepository测试类"""
    
    @pytest.fixture
    def repository(self, db):
        """创建Repository实例"""
        return TaskRepository(db)
    
    @pytest.fixture
    def sample_task(self):
        """创建示例任务"""
        return Task(
            id='test-001',
            file_path='test.mp3',
            file_name='test.mp3',
            file_size=1024,
            file_hash='abc123',
            duration=60.0,
            status='pending'
        )
    
    def test_create_task(self, repository, sample_task):
        """测试创建任务"""
        created = repository.create(sample_task)
        assert created.id == 'test-001'
        assert created.status == 'pending'
    
    def test_find_by_id(self, repository, sample_task):
        """测试根据ID查找任务"""
        repository.create(sample_task)
        found = repository.find_by_id('test-001')
        assert found is not None
        assert found.file_name == 'test.mp3'
    
    def test_update_status(self, repository, sample_task):
        """测试更新任务状态"""
        repository.create(sample_task)
        task = repository.find_by_id('test-001')
        task.status = 'transcribing'
        repository.update(task)
        
        updated = repository.find_by_id('test-001')
        assert updated.status == 'transcribing'
    
    def test_delete_task(self, repository, sample_task):
        """测试删除任务"""
        repository.create(sample_task)
        repository.delete('test-001')
        
        found = repository.find_by_id('test-001')
        assert found is None
```

## 4. 测试执行

### 4.1 测试配置
```ini
# pytest.ini
[pytest]
testpaths = tests
python_files = test_*.py
python_classes = Test*
python_functions = test_*
addopts = 
    -v
    --tb=short
    --strict-markers
    --cov=src
    --cov-report=html
    --cov-report=term-missing
markers =
    slow: marks tests as slow
    benchmark: marks tests as benchmark tests
    integration: marks tests as integration tests
```

### 4.2 测试命令
```bash
# 运行所有测试
pytest

# 运行单元测试
pytest tests/unit/

# 运行集成测试
pytest tests/integration/

# 运行性能测试
pytest --benchmark-only

# 生成覆盖率报告
pytest --cov=src --cov-report=html
```

## 5. 持续集成

### 5.1 GitHub Actions配置
```yaml
# .github/workflows/test.yml
name: Tests

on:
  push:
    branches: [ main, develop ]
  pull_request:
    branches: [ main ]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ['3.11', '3.12']
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Set up Python ${{ matrix.python-version }}
      uses: actions/setup-python@v4
      with:
        python-version: ${{ matrix.python-version }}
    
    - name: Install dependencies
      run: |
        python -m pip install --upgrade pip
        pip install -r requirements.txt
        pip install -r requirements-dev.txt
    
    - name: Run linting
      run: |
        flake8 src tests
    
    - name: Run type checking
      run: |
        mypy src
    
    - name: Run tests
      run: |
        pytest --cov=src --cov-report=xml
```
