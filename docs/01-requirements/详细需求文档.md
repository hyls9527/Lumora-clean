# 详细需求文档 — Lumora

> **审计原则**：每个用户故事必须追溯到具体代码（Store action、API 函数、组件）。

## 1. 图片管理模块

### 1.1 图片导入

**用户故事**：作为 AI 图片创作者，我想批量导入文件夹中的图片，以便统一管理我的作品。

**验收标准**：
- [ ] 支持拖拽文件/文件夹到窗口自动导入
- [ ] 支持通过对话框选择文件夹导入
- [ ] 自动提取 PNG 文本块（prompt、seed、model 等元数据）
- [ ] 导入后显示导入统计（新增/跳过/失败数量）
- [ ] 支持的格式：jpg, jpeg, png, gif, webp, bmp, tiff

**代码依据**：
- Store: `imageStore.importImages()`
- API: `api.importImages(folderPath)`
- IPC: `import_images`
- 组件: `ImportPage.tsx`, `DropOverlay.tsx`

### 1.2 图片列表

**用户故事**：作为用户，我想浏览我的图片库，支持分页加载和多种排序方式。

**验收标准**：
- [ ] 默认按时间倒序排列
- [ ] 支持按评分、模型、文件大小排序
- [ ] 支持 Creator Mode（显示 prompt/seed/model）和 Normal Mode（简洁显示）
- [ ] 支持 Grid View 和 List View 切换
- [ ] 无限滚动加载（每页 40 张）
- [ ] 支持按模型过滤

**代码依据**：
- Store: `imageStore.fetchImages()`, `imageStore.loadMore()`, `imageStore.setSortBy()`
- IPC: `list_images`
- 组件: `GalleryPage.tsx`, `ImageCard.tsx`, `LazyLoad.tsx`

### 1.3 图片搜索

**用户故事**：作为用户，我想通过关键词搜索图片，支持全文搜索和语义搜索。

**验收标准**：
- [ ] 全文搜索（FTS5）：搜索文件路径、metadata_json
- [ ] 高级搜索：按字段（prompt/seed/model/negative_prompt/sampler）搜索
- [ ] 语义搜索：输入文本描述，返回相似图片
- [ ] 以图搜图：选择一张图片，找到相似图片
- [ ] 搜索结果按相似度排序
- [ ] 可调节相似度阈值（0-100）

**代码依据**：
- Store: `imageStore.searchImages()`, `semanticSearchStore`
- API: `api.searchImagesAdvanced()`, `api.searchSemantic()`
- IPC: `search_images`, `search_images_advanced`, `search_semantic_cmd`
- 组件: `SearchPage.tsx`, `SemanticSearchBar.tsx`

### 1.4 评分与收藏

**用户故事**：作为用户，我想给图片评分和收藏，以便快速找到优质图片。

**验收标准**：
- [ ] 支持 0-5 星评分
- [ ] 点击收藏图标切换收藏状态
- [ ] 收藏列表独立页面
- [ ] 评分和收藏状态实时同步到后端

**代码依据**：
- Store: `imageStore.updateImage()`
- IPC: `update_rating`, `toggle_favorite`, `list_favorites`
- 组件: `Rating.tsx`, `FavoritesPage.tsx`
