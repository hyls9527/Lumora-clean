# 概要需求文档 — Lumora v0.5.1

## 1. 产品定位

Lumora 是一个桌面端图片管理工具，面向使用 Stable Diffusion、ComfyUI 等 AI 绘图工具的创作者。核心解决"本地大量生成图片难以检索和管理"的问题。

## 2. 目标用户

AI 绘图创作者，拥有本地数百至数千张生成图片，需要按内容、prompt、模型等维度检索和管理。

## 3. 核心功能域

| 功能域 | 优先级 | 说明 |
|--------|--------|------|
| 图片导入 | P0 | 文件夹/文件拖拽导入，自动提取元数据 |
| 图片浏览 | P0 | 网格/列表视图，分页加载 |
| 搜索 | P0 | FTS5 全文搜索 + 字段范围搜索 |
| 标签 | P0 | 创建标签，关联图片，批量操作 |
| 回收站 | P0 | 软删除/恢复/永久删除，批量操作 |
| 评分收藏 | P1 | 0-5 评分 + 收藏标记 |
| AI 分析 | P1 | Ollama llava 分析图片内容 |
| 语义搜索 | P1 | 自然语言/图片到图片相似搜索 |
| 变体追踪 | P2 | 同 prompt 不同 seed 自动分组 |
| 导出 | P1 | 选中图片导出 + 自定义文件名模板 |
| 数据库备份 | P1 | SQLite 文件导入/导出 |
| 仪表盘 | P2 | 图片统计概览 |
| 设置 | P0 | 语言（中/英）、主题（明/暗） |
| 快捷键 | P1 | 命令面板 ⌘K + 键盘导航 |
| 自动更新 | P2 | GitHub Releases 检测 + 一键更新 |

## 4. 非功能性需求

| 维度 | 要求 |
|------|------|
| 平台 | Windows / macOS / Linux |
| 性能 | 1000 张图片批量导入可完成（有测试验证） |
| 隐私 | 数据存储于本地，无遥测，无云同步 |
| 安全 | CSP 限制外部连接，仅允许 localhost + GitHub + Google Fonts |
| 可用性 | 大部分操作支持键盘（侧边栏导航除外） |

## 5. 技术约束

- 前端：React 19 + TypeScript + Tailwind CSS v4 + Zustand 5
- 后端：Rust (Tauri 2) + SQLite (FTS5 + sqlite-vec)
- AI：依赖本地 Ollama 服务（可选，不影响基础功能）
- 设计语言：古卷·灯火（见 DESIGN.md）

## 6. 范围外

以下功能明确不在当前版本范围内：
- 云同步/多设备同步
- 在线图库/分享
- 图片编辑/修图
- 视频管理
- 移动端适配
