# 概要需求文档 — Lumora

> **审计原则**：每个功能点必须追溯到具体代码（IPC 命令、组件、Store）。

## 1. 产品定位

Lumora 是一个本地优先的 AI 图片管理桌面应用，面向 AI 图片创作者（Stable Diffusion / ComfyUI / Flux 用户）。

**技术栈**：Tauri 2 + React 19 + TypeScript + Tailwind CSS v4 + Rust + SQLite + Ollama

## 2. 功能模块概览

| 模块 | 功能 | IPC 命令 | 前端页面 |
|------|------|----------|----------|
| 图片管理 | 导入、列表、搜索、评分、收藏 | `import_images`, `list_images`, `search_images`, `update_rating`, `toggle_favorite`, `list_favorites` | Gallery, Import |
| 标签系统 | 创建、删除、关联图片 | `create_tag`, `list_tags`, `delete_tag`, `update_tag`, `add_tag_to_image`, `remove_tag_from_image`, `get_image_tags` | Tags |
| 回收站 | 软删除、恢复、永久删除、批量操作 | `soft_delete_image`, `restore_image`, `permanent_delete_image`, `list_trash`, `empty_trash`, `batch_soft_delete`, `batch_restore`, `batch_permanent_delete`, `batch_add_tag`, `batch_remove_tag` | Trash |
| AI 分析 | 图片分析、标签建议、分析历史 | `analyze_image_cmd`, `get_analysis_result_cmd`, `get_analysis_history_cmd`, `apply_ai_tags_cmd` | DetailModal |
| 语义搜索 | 嵌入生成、向量搜索、相似图片 | `generate_embedding`, `get_embedding_status_cmd`, `search_semantic_cmd`, `get_embedding_stats_cmd`, `embed_text_cmd`, `generate_embedding_for_image_cmd` | Search |

| 仪表盘 | 统计数据 | `get_dashboard_stats` | Dashboard |

| 备份 | 数据库导入导出 | `export_database`, `import_database` | Settings |

| Ollama 集成 | 状态检查、主机配置 | `check_ollama_status`, `get_ollama_host` | Settings |

## 3. 核心数据模型

| 表名 | 用途 | 关键字段 |
|------|------|----------|
| `images` | 图片主表 | id, file_path, file_hash, width, height, format, rating, favorite, variant_group_id |
| `tags` | 标签表 | id, name, color |
| `image_tags` | 图片-标签关联 | image_id, tag_id |
| `embeddings` | 嵌入元数据 | image_id, embedding, dimensions, status |
| `vec_embeddings` | 向量索引 (sqlite-vec) | image_id, embedding float[768] |
| `analysis_history` | AI 分析历史 | id, image_id, result_json |
| `variant_groups` | 变体组 | id, prompt |
| `images_fts` | FTS5 全文索引 | file_path, metadata_json |

## 4. 非功能需求

| 需求 | 实现方式 | 代码依据 |
|------|----------|----------|
| 本地优先 | SQLite 本地存储，无云同步 | `db/mod.rs` |
| 隐私保护 | CSP 限制，仅本地通信 | `tauri.conf.json` |
| 暗色主题 | CSS 变量 + data-theme 属性 | `tokens.ts` |
| 国际化 | i18n JSON 文件 | `src/i18n/` |
| 自动更新 | GitHub Releases + minisign | `.github/workflows/release.yml` |
| 性能优化 | LazyLoad + CSS containment + LRU 缓存 | `semanticCache.ts`, `LazyLoad.tsx` |
