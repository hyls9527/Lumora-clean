## 3. 环境搭建

### 3.1 Python环境搭建

#### 3.1.1 安装Python
```bash
# 下载Python 3.11/3.12
# https://www.python.org/downloads/

# 安装Python（勾选Add to PATH）
# 验证安装
python --version
# Python 3.11.x 或 3.12.x

# 验证pip
pip --version
```

#### 3.1.2 创建虚拟环境
```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境（Windows）
.venv\Scripts\activate

# 验证虚拟环境
python --version
pip --version
```

### 3.2 CUDA环境搭建（可选）

#### 3.2.1 安装CUDA Toolkit
```bash
# 下载CUDA Toolkit 11.8/12.1
# https://developer.nvidia.com/cuda-toolkit-archive

# 安装CUDA Toolkit
# 验证安装
nvcc --version
```

#### 3.2.2 验证CUDA安装
```bash
# 验证CUDA
nvidia-smi

# 验证PyTorch CUDA支持
python -c "import torch; print(torch.cuda.is_available())"
```

### 3.3 FFmpeg环境搭建

#### 3.3.1 下载FFmpeg
```bash
# 下载FFmpeg
# https://ffmpeg.org/download.html

# 解压到目录
# 例如：C:\ffmpeg
```

#### 3.3.2 配置环境变量
```bash
# 添加FFmpeg到PATH
# 系统属性 -> 环境变量 -> Path -> 新建
# 添加：C:\ffmpeg\bin

# 验证安装
ffmpeg -version
ffprobe -version
```

## 4. 应用部署

### 4.1 获取代码

#### 4.1.1 克隆代码仓库
```bash
# 克隆代码
git clone https://github.com/your-repo/lingjian.git
cd lingjian

# 切换到v2.0分支
git checkout v2.0
```

### 4.2 安装依赖

#### 4.2.1 安装核心依赖
```bash
# 激活虚拟环境
.venv\Scripts\activate

# 安装核心依赖
pip install -r requirements.txt

# 验证安装
pip list
```

### 4.3 配置应用

#### 4.3.1 创建配置文件
```bash
# 创建配置目录
mkdir config

# 创建配置文件
cat > config/config.toml << EOF
[batch]
max_workers = 2
gpu_memory_limit = 6.0
auto_adjust = true

[export]
default_format = "docx"
include_timestamp = true
include_speaker = true
add_watermark = false
desensitize_level = "L2"

[transcription]
model_size = "medium"
language = "auto"
beam_size = 5

[logging]
level = "INFO"
file = "logs/app.log"
max_size = "10 MB"
backup_count = 5
EOF
```

### 4.4 下载模型

#### 4.4.1 下载Whisper模型
```bash
# 创建模型目录
mkdir -p models/faster-whisper

# 下载模型（首次运行自动下载）
python -c "
from faster_whisper import WhisperModel
model = WhisperModel('medium', device='cuda', compute_type='int8_float16')
print('Model downloaded successfully')
"
```

## 5. 应用启动

### 5.1 启动应用程序

#### 5.1.1 命令行启动
```bash
# 激活虚拟环境
.venv\Scripts\activate

# 启动应用程序
python -m lingjian

# 或者
python src/lingjian/__main__.py
```

#### 5.1.2 创建启动脚本
```bash
# 创建启动脚本
cat > start.bat << EOF
@echo off
cd /d %~dp0
call .venv\Scripts\activate
python -m lingjian
pause
EOF
```

#### 5.1.3 验证启动
```bash
# 启动应用程序
start.bat

# 检查日志
type logs\app.log
```

## 6. 配置管理

### 6.1 配置文件位置
```
lingjian/
├── config/
│   ├── config.toml          # 主配置文件
│   └── logging.toml         # 日志配置
├── data/
│   └── lingjian.db          # 数据库文件
├── logs/
│   ├── app.log              # 应用日志
│   └── error.log            # 错误日志
└── models/                  # 模型目录
```

### 6.2 配置文件详解

#### 6.2.1 主配置文件 (config.toml)
```toml
[batch]
# 最大并发任务数（1-4）
max_workers = 2

# GPU显存限制（GB）
gpu_memory_limit = 6.0

# 自动调整并发数
auto_adjust = true

[export]
# 默认导出格式
default_format = "docx"

# 导出选项
include_timestamp = true
include_speaker = true

# 脱敏设置
desensitize = false
desensitize_level = "L2"

[transcription]
# 模型大小：tiny/base/small/medium/large-v3
model_size = "medium"

# 语言：auto/zh/en
language = "auto"

# Beam search大小
beam_size = 5

[logging]
# 日志级别：DEBUG/INFO/WARNING/ERROR
level = "INFO"

# 日志文件
file = "logs/app.log"

# 日志轮转
max_size = "10 MB"
backup_count = 5
```

## 7. 监控与维护

### 7.1 日志监控

#### 7.1.1 查看实时日志
```bash
# 实时查看日志
tail -f logs\app.log

# 查看错误日志
tail -f logs\error.log

# 搜索特定日志
findstr "ERROR" logs\app.log
```

### 7.2 性能监控

#### 7.2.1 系统监控
```bash
# 监控CPU和内存
tasklist /fi "imagename eq python.exe"

# 监控GPU
nvidia-smi -l 1

# 监控磁盘
wmic logicaldisk get size,freespace,caption
```

### 7.3 维护操作

#### 7.3.1 数据库维护
```bash
# 清理旧数据
sqlite3 data\lingjian.db "DELETE FROM tasks WHERE status = 'completed' AND completed_at < datetime('now', '-30 days');"

# 重建索引
sqlite3 data\lingjian.db "REINDEX;"

# 压缩数据库
sqlite3 data\lingjian.db "VACUUM;"
```

## 8. 故障排查

### 8.1 常见问题

#### 8.1.1 启动失败
**问题**: 应用程序无法启动
**排查**:
```bash
# 检查Python版本
python --version

# 检查依赖
pip list | findstr PySide6

# 检查日志
type logs\app.log | findstr "Error"

# 检查配置
type config\config.toml
```

#### 8.1.2 模型加载失败
**问题**: 模型加载失败
**排查**:
```bash
# 检查模型目录
dir models\faster-whisper

# 检查网络连接（首次下载需要）
ping huggingface.co

# 检查磁盘空间
wmic logicaldisk get size,freespace,caption

# 检查显存
nvidia-smi
```

#### 8.1.3 转写失败
**问题**: 音频转写失败
**排查**:
```bash
# 检查音频文件
ffprobe test.mp3

# 检查日志
type logs\app.log | findstr "transcribe"

# 检查GPU显存
nvidia-smi

# 检查CUDA版本
nvcc --version
```
