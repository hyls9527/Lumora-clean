# Lumora 产品路线图 (v0.8.0 → v1.0)

> **更新时间**：2026-07-13
> **当前版本**：v0.7.0 (445 前端测试 + 59 Rust 测试)
> **目标用户**：AI 图片创作者 (Stable Diffusion / ComfyUI / Flux)

---

## 产品愿景

**一句话**：让 AI 图片创作者在海量生成图中快速找到、组织、理解、复用他们的作品。

**核心价值主张**：
1. 🔍 **找得到** — 语义搜索 + 以图搜图 + 全文搜索
2. 🏷️ **管得好** — AI 自动标签 + 变体分组 + 智能收藏
3. 🎨 **看得懂** — AI 分析 prompt/模型/参数 + 色彩提取
4. 📦 **导得出** — 批量导出 + 模板命名 + 格式转换

---

## 版本规划

### v0.8.0 — 体验打磨 (Polish & Performance)

**主题**：把现有功能做到极致，解决用户痛点

| 功能 | 优先级 | 说明 | 预估工作量 |
|------|--------|------|-----------|
| **变体对比视图** | P0 | 同 prompt 不同 seed 的图片并排对比 | 3 天 |
| **高级筛选面板** | P0 | 组合条件：模型 + 日期范围 + 评分 + 尺寸 | 2 天 |
| **批量重命名** | P1 | 按模板批量重命名文件（导出前处理） | 1 天 |
| **图片信息增强** | P1 | 显示完整 SD 参数（Steps, CFG, Sampler, Seed） | 1 天 |
| **性能优化** | P1 | 10K+ 图片场景下的虚拟滚动优化 | 2 天 |
| **键盘快捷键完善** | P2 | 全局快捷键 + 快捷键帮助面板 | 1 天 |
| **错误恢复** | P2 | 图片加载失败自动重试 + 离线状态提示 | 1 天 |

**验收标准**：
- [ ] 变体组图片可并排对比（2-6 张）
- [ ] 筛选面板支持 3+ 条件组合
- [ ] 10K 图片库滚动帧率 ≥ 30fps
- [ ] 所有现有测试通过 + 新增 30+ 测试

### v0.9.0 — 能力扩展 (Capability Expansion)

**主题**：增加用户高频需求的新能力

| 功能 | 优先级 | 说明 | 预估工作量 |
|------|--------|------|-----------|

| **批量格式转换** | P1 | PNG↔WebP↔JPEG 批量转换 | 2 天 |

| **收藏夹增强** | P1 | 多收藏夹 + 嵌套分类 | 2 天 |
| **Prompt 模板库** | P2 | 保存常用 prompt 片段，快速复用 | 1 天 |
| **图片水印** | P2 | 批量添加文字/图片水印 | 2 天 |
| **EXIF 编辑** | P2 | 编辑图片元数据（非破坏性） | 2 天 |

**验收标准**：
- [ ] 编辑操作可撤销（Ctrl+Z）
- [ ] 批量转换 100 张图片 < 30 秒
- [ ] 智能相册支持 5+ 规则类型
- [ ] 新增 50+ 测试

## 依赖关系

```
v0.8.0 (体验打磨)
  ├── 变体对比视图 ← 依赖现有 variant_groups
  ├── 高级筛选 ← 依赖现有 search_images_advanced
  └── 性能优化 ← 依赖现有 LazyLoad + VirtualGrid

v0.9.0 (能力扩展)
  ├── 图片编辑 ← 依赖 Rust image crate
  ├── 批量转换 ← 依赖现有 export_images
  └── 智能相册 ← 依赖现有 tag + rating 系统

v1.0 (生产就绪)
  ├── 插件系统 ← 依赖 Tauri plugin API
  ├── 多 AI 后端 ← 依赖现有 Ollama 集成
  └── 数据迁移 ← 依赖现有 import_images
```
