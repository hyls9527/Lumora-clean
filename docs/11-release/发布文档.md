## 4. 发布流程

### 4.1 代码准备

#### 4.1.1 代码冻结
```bash
# 创建发布分支
git checkout -b release/v2.0.0

# 代码冻结，禁止新功能提交
# 只允许Bug修复
```

#### 4.1.2 代码审查
```bash
# 执行代码审查
# 检查代码风格
flake8 src tests

# 检查类型
mypy src

# 检查安全漏洞
bandit -r src
```

#### 4.1.3 代码合并
```bash
# 合并到主分支
git checkout main
git merge release/v2.0.0

# 推送到远程
git push origin main
```

### 4.2 测试执行

#### 4.2.1 自动化测试
```bash
# 运行所有测试
pytest --cov=src --cov-report=html

# 运行性能测试
pytest --benchmark-only

# 运行集成测试
pytest tests/integration/

# 生成测试报告
pytest --html=report.html
```

### 4.3 构建发布包

#### 4.3.1 创建发布目录
```bash
# 创建发布目录
mkdir release/v2.0.0

# 复制源代码
xcopy /E /I src release/v2.0.0/src
xcopy /E /I tests release/v2.0.0/tests
xcopy /E /I docs release/v2.0.0/docs

# 复制配置文件
copy pyproject.toml release/v2.0.0/
copy requirements.txt release/v2.0.0/
copy README.md release/v2.0.0/
copy LICENSE release/v2.0.0/
```

#### 4.3.2 创建安装脚本
```bash
# 创建安装脚本
cat > release/v2.0.0/install.bat << EOF
@echo off
echo 安装聆鉴 v2.0.0
echo.

REM 检查Python版本
python --version
if errorlevel 1 (
    echo 错误：未找到Python
    pause
    exit /b 1
)

REM 创建虚拟环境
echo 创建虚拟环境...
python -m venv .venv
call .venv\Scripts\activate

REM 安装依赖
echo 安装依赖...
pip install -r requirements.txt

echo.
echo 安装完成！
echo 运行 start.bat 启动应用程序
pause
EOF

# 创建启动脚本
cat > release/v2.0.0/start.bat << EOF
@echo off
cd /d %~dp0
call .venv\Scripts\activate
python -m lingjian
pause
EOF
```

#### 4.3.3 打包发布包
```bash
# 创建ZIP包
cd release/v2.0.0
7z a -tzip ../lingjian-2.0.0-win64.zip *

# 创建TAR.GZ包
tar -czvf ../lingjian-2.0.0.tar.gz *

# 计算哈希值
sha256sum ../lingjian-2.0.0-win64.zip > ../lingjian-2.0.0-win64.zip.sha256
sha256sum ../lingjian-2.0.0.tar.gz > ../lingjian-2.0.0.tar.gz.sha256
```

## 7. 回滚计划

### 7.1 回滚触发条件
- 严重Bug影响核心功能
- 性能严重下降
- 安全漏洞
- 数据损坏

### 7.2 回滚流程

#### 7.2.1 代码回滚
```bash
# 回滚到上一个版本
git checkout v1.0.0

# 创建回滚分支
git checkout -b rollback/v2.0.0

# 回滚代码
git revert v2.0.0

# 提交回滚
git commit -m "rollback: revert v2.0.0"
git tag -a v2.0.0-rollback -m "Rollback v2.0.0"
git push origin rollback/v2.0.0
git push origin v2.0.0-rollback
```

#### 7.2.2 数据库回滚
```bash
# 恢复数据库备份
copy backups\pre-v2.0.0\lingjian.db data\lingjian.db

python -m lingjian.migrate --rollback
```
