| 特性 | 值 |

## 2. Schema 定义

### 2.1 V1 — 图片表

#### images 表

```sql
CREATE TABLE IF NOT EXISTS images (
    id            TEXT PRIMARY KEY,
    file_path     TEXT NOT NULL UNIQUE,
    file_hash     TEXT NOT NULL,
    file_size_kb  INTEGER NOT NULL,
    width         INTEGER,
    height        INTEGER,
    format        TEXT NOT NULL,
    created_at    TEXT NOT NULL,
    imported_at   TEXT NOT NULL DEFAULT (datetime('now')),
    deleted       INTEGER DEFAULT 0,
    rating        INTEGER DEFAULT 0,
    favorite      INTEGER DEFAULT 0,
    metadata_json TEXT
);
```

**字段说明**：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PRIMARY KEY | UUID |
| file_path | TEXT | NOT NULL, UNIQUE | 文件绝对路径 |
| file_hash | TEXT | NOT NULL | 文件哈希（防重复导入） |
| file_size_kb | INTEGER | NOT NULL | 文件大小 (KB) |
| width | INTEGER | NULL | 图片宽度 |
| height | INTEGER | NULL | 图片高度 |
| format | TEXT | NOT NULL | 图片格式 (png/jpg/webp) |
| created_at | TEXT | NOT NULL | 文件创建时间 |
| imported_at | TEXT | NOT NULL | 导入时间 |
| deleted | INTEGER | DEFAULT 0 | 软删除标记 (0/1) |
| rating | INTEGER | DEFAULT 0 | 评分 (0-5) |
| favorite | INTEGER | DEFAULT 0 | 收藏标记 (0/1) |
| metadata_json | TEXT | NULL | 元数据 JSON (prompt/seed/model) |

**代码依据**：`src-tauri/src/db/schema.rs` V1_CREATE_IMAGES

---

#### images_fts 虚表 (FTS5)

```sql
CREATE VIRTUAL TABLE IF NOT EXISTS images_fts USING fts5(
    file_path,
    metadata_json,
    content='images',
    content_rowid='rowid'
);
```

**用途**：全文搜索文件路径和元数据

**代码依据**：`src-tauri/src/db/schema.rs` V1_FTS5

---

#### 索引

```sql
-- 文件哈希索引（防重复导入）
CREATE INDEX IF NOT EXISTS idx_images_file_hash ON images(file_hash);

-- 评分索引（按评分查询）
CREATE INDEX IF NOT EXISTS idx_images_rating ON images(rating) WHERE deleted = 0;

-- 创建时间索引（按时间查询）
CREATE INDEX IF NOT EXISTS idx_images_created ON images(created_at) WHERE deleted = 0;
```

**代码依据**：`src-tauri/src/db/schema.rs` V1_INDEX_*

---

#### 触发器

```sql
-- INSERT 后同步到 FTS5
CREATE TRIGGER IF NOT EXISTS images_ai AFTER INSERT ON images BEGIN
    INSERT INTO images_fts(rowid, file_path, metadata_json)
    VALUES (new.rowid, new.file_path, new.metadata_json);
END;

-- DELETE 后从 FTS5 删除
CREATE TRIGGER IF NOT EXISTS images_ad AFTER DELETE ON images BEGIN
    INSERT INTO images_fts(images_fts, rowid, file_path, metadata_json)
    VALUES ('delete', old.rowid, old.file_path, old.metadata_json);
END;

-- UPDATE 后更新 FTS5
CREATE TRIGGER IF NOT EXISTS images_au AFTER UPDATE ON images BEGIN
    INSERT INTO images_fts(images_fts, rowid, file_path, metadata_json)
    VALUES ('delete', old.rowid, old.file_path, old.metadata_json);
    INSERT INTO images_fts(rowid, file_path, metadata_json)
    VALUES (new.rowid, new.file_path, new.metadata_json);
END;
```

**代码依据**：`src-tauri/src/db/schema.rs` V1_TRIGGER_*

### 2.2 V2 — 标签表

#### tags 表

```sql
CREATE TABLE IF NOT EXISTS tags (
    id         TEXT PRIMARY KEY,
    name       TEXT NOT NULL UNIQUE,
    color      TEXT,
    created_at TEXT NOT NULL DEFAULT (datetime('now'))

```

**字段说明**：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PRIMARY KEY | UUID |
| name | TEXT | NOT NULL, UNIQUE | 标签名称 |
| color | TEXT | NULL | 标签颜色 (HEX) |
| created_at | TEXT | NOT NULL | 创建时间 |

**代码依据**：`src-tauri/src/db/schema.rs` V2_CREATE_TAGS

---

#### image_tags 关联表

```sql
CREATE TABLE IF NOT EXISTS image_tags (
    image_id TEXT NOT NULL REFERENCES images(id),
    tag_id   TEXT NOT NULL REFERENCES tags(id),
    PRIMARY KEY (image_id, tag_id)
);
```

**字段说明**：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| image_id | TEXT | NOT NULL, FK → images.id | 图片 ID |
| tag_id | TEXT | NOT NULL, FK → tags.id | 标签 ID |

**复合主键**：`(image_id, tag_id)`

**代码依据**：`src-tauri/src/db/schema.rs` V2_CREATE_IMAGE_TAGS

---

#### 索引

### 2.4 V4 — 嵌入表

#### embeddings 表

```sql
CREATE TABLE IF NOT EXISTS embeddings (
    image_id     TEXT PRIMARY KEY REFERENCES images(id),
    embedding    BLOB NOT NULL,
    dimensions   INTEGER NOT NULL DEFAULT 512,
    status       TEXT NOT NULL DEFAULT 'embedded' CHECK(status IN ('embedded', 'pending', 'error')),
    generated_at TEXT NOT NULL DEFAULT (datetime('now'))
);
```

**字段说明**：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| image_id | TEXT | PRIMARY KEY, FK → images.id | 图片 ID |
| embedding | BLOB | NOT NULL | 嵌入向量 |
| dimensions | INTEGER | NOT NULL, DEFAULT 512 | 向量维度 |
| status | TEXT | NOT NULL, CHECK | 状态 (embedded/pending/error) |
| generated_at | TEXT | NOT NULL | 生成时间 |

**代码依据**：`src-tauri/src/db/schema.rs` V4_CREATE_EMBEDDINGS

---

#### vec_embeddings 虚表 (sqlite-vec)

```sql
CREATE VIRTUAL TABLE IF NOT EXISTS vec_embeddings USING vec0(
    image_id TEXT PRIMARY KEY,
    embedding float[768]
);
```

**用途**：向量相似度搜索

**参数**：
- 向量维度：768
- 距离度量：默认 (cosine)

**代码依据**：`src-tauri/src/db/schema.rs` V4_CREATE_VEC_TABLE

## 4. 数据字典

### 4.1 metadata_json 格式

```json
{
  "prompt": "a cat sitting on a chair",
  "negative_prompt": "blurry, low quality",
  "seed": 1234567890,
  "model": "sdxl",
  "sampler": "euler_a",
  "steps": 20,
  "cfg_scale": 7.0,
  "width": 1024,
  "height": 1024
}
```

**来源**：PNG 文本块解析

**代码依据**：`src-tauri/src/metadata/sd.rs`

---

### 4.2 result_json 格式 (AI 分析)

```json
{
  "description": "A cat sitting on a wooden chair...",
  "tags": ["cat", "chair", "indoor", "pet"],
  "confidence": 0.85
}
```

**来源**：Ollama llava 模型

**代码依据**：`src-tauri/src/commands/ai.rs`

---

### 4.3 embedding 格式

- 类型：BLOB
- 格式：f32 数组
- 维度：768 (nomic-embed-text)
- 字节序：Little-endian

**代码依据**：`src-tauri/src/commands/embeddings.rs`

## 6. 查询模式

### 6.1 常用查询

| 查询 | SQL | 代码位置 |
|------|------|----------|
| 分页列表 | `SELECT * FROM images WHERE deleted = 0 ORDER BY created_at DESC LIMIT ? OFFSET ?` | `commands/images/ops.rs` |
| 全文搜索 | `SELECT * FROM images_fts WHERE images_fts MATCH ?` | `commands/images/search.rs` |
| 向量搜索 | `SELECT * FROM vec_embeddings WHERE embedding MATCH ? ORDER BY distance LIMIT ?` | `commands/embeddings.rs` |
| 标签查询 | `SELECT t.* FROM tags t JOIN image_tags it ON t.id = it.tag_id WHERE it.image_id = ?` | `commands/tags.rs` |
| 回收站 | `SELECT * FROM images WHERE deleted = 1 ORDER BY deleted_at DESC` | `commands/trash.rs` |
