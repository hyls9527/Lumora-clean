## 关系说明

| 关系 | 类型 | 外键 | 级联行为 |
|------|------|------|---------|
| images → image_tags | 1:N | image_tags.image_id | 永久删除时级联清理 |
| tags → image_tags | 1:N | image_tags.tag_id | 删除标签时清理关联 |
| images → embeddings | 1:1 | embeddings.image_id | 永久删除时级联清理 |
| images → vec_embeddings | 1:1 | vec_embeddings.image_id | 永久删除时级联清理（日志警告） |
| images → analysis_history | 1:N | analysis_history.image_id | 永久删除时级联清理 |
| images → variant_groups | N:1 | images.variant_group_id | 无级联 |

## 迁移历史

| 版本 | 内容 | 对应代码 |
|------|------|---------|
| v1 | images 表 + FTS5 全文索引 | `V1_CREATE_IMAGES`、`V1_FTS5` |
| v2 | tags 表 + image_tags 关联表 | `V2_CREATE_TAGS`、`V2_CREATE_IMAGE_TAGS` |
| v3 | images 新增 deleted_at 列 | `V3_ADD_DELETED_AT` |
| v4 | embeddings 表 + vec_embeddings 向量表 | `V4_CREATE_EMBEDDINGS` |
| v5 | analysis_history 表 | `V5_CREATE_ANALYSIS_HISTORY` |
| v6 | variant_groups 表 + images.variant_group_id | `V6_CREATE_VARIANT_GROUPS` |

## 索引策略

| 索引 | 类型 | 用途 |
|------|------|------|
| images.file_path | UNIQUE | 防止重复导入 |
| images.file_hash | （无索引，应用层去重） | — |
| images_fts | FTS5 | 全文搜索 file_path + metadata_json |
| vec_embeddings | vec0 | 向量相似度搜索（cosine distance） |
| image_tags(image_id, tag_id) | 复合主键 | 标签关联查询 |

## 数据规模预估

当前版本无大规模数据测试。`bulk_insert_1000_images_performance` 测试验证了 1000 张图片的批量导入性能。
