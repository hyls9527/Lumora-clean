### 2.2 状态管理模式

#### Store 结构模板

```typescript
interface XxxStore {
  // State
  data: DataType[];
  loading: boolean;
  error: string | null;
  
  // Async Actions
  fetchData: () => Promise<void>;
  
  // Sync Actions
  setFilter: (filter: FilterType) => void;
}
```

#### 乐观更新模式

```typescript
// trashStore.ts 示例
restoreImage: async (id: string) => {
  // 1. 乐观更新
  const prev = get().images;
  set({ images: prev.filter(img => img.id !== id) });
  
  try {
    // 2. API 调用
    await api.restoreImage(id);
  } catch (err) {
    // 3. 失败回滚
    set({ images: prev });
  }
}
```

### 3.2 错误处理架构

#### AppError 枚举

```rust
pub enum AppError {
    Db(String),          // 数据库错误
    Io(String),          // 文件 I/O 错误
    NotFound(String),    // 记录未找到
    InvalidInput(String), // 输入验证失败
    External(String),    // 外部服务错误
    Lock,                // 锁失败
}
```

#### 错误转换链

```
rusqlite::Error → AppError::Db
std::io::Error → AppError::Io
reqwest::Error → AppError::External
serde_json::Error → AppError::External
String → AppError::External
```
