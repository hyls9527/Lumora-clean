## 2. 前端 Store 详细设计

### 2.1 ImageStore 状态机

**状态**：
```typescript
interface ImageStore {
  images: ImageRecord[];
  filters: FilterState;
  loading: boolean;
  error: string | null;
  page: number;
  total: number;
  perPage: number;
}
```

**FilterState**：
```typescript
interface FilterState {
  mode: 'creator' | 'normal';
  view: 'grid' | 'list';
  sortBy: 'time' | 'rating' | 'model' | 'size';
  modelFilter: string;
  searchQuery: string;
  searchField: string;
  searchMode: 'text' | 'image';
  similarityThreshold: number;
}
```

**状态转换**：
```
fetchImages(page) → loading=true → API 调用 → loading=false, 更新 images/total
loadMore() → loading=true → API 调用 → loading=false, 追加 images
searchImages(query) → loading=true → API 调用 → loading=false, 替换 images
setSortBy(sortBy) → 更新 filters.sortBy → 自动重新排序
```

**代码依据**：`src/stores/imageStore.ts`

### 2.2 TrashStore 乐观更新

**乐观更新模式**：
```typescript
restoreImage: async (id: string) => {
  // 1. 乐观更新：立即从列表移除
  const prev = get().images;
  set({ images: prev.filter(img => img.id !== id), total: get().total - 1 });
  
  try {
    // 2. 调用 API
    await api.restoreImage(id);
  } catch (err) {
    // 3. 失败回滚
    set({ images: prev, total: get().total + 1 });
    set({ error: err.message });
  }
}
```

## 3. 缓存策略详细设计

### 3.1 SemanticCache

**缓存结构**：
```typescript
interface CachedEntry {
  results: { id: string; similarity: number }[];
  ts: number;  // 时间戳
}
```

**配置参数**：
- TTL: 30 分钟 (`TTL_MS = 1000 * 60 * 30`)
- 最大条目: 200 (`MAX_ENTRIES = 200`)
- 持久化: localStorage (`CACHE_KEY = 'lumora:semantic-cache'`)
- 防抖延迟: 300ms

**LRU 实现**：
```typescript
function getCachedResult(query: string) {
  hydrate();
  const key = normalizeQuery(query);
  const entry = cache.get(key);
  if (!entry) return null;
  if (isExpired(entry)) {
    cache.delete(key);
    persist();
    return null;
  }
  // LRU: 移到末尾
  cache.delete(key);
  cache.set(key, entry);
  return entry.results;
}
```

**失效策略**：
- TTL 过期自动删除
- 写命令触发清除（`onWriteCommand()`）
- 手动清除

**代码依据**：`src/lib/api/semanticCache.ts`
