## 3. 数据流

### 3.1 图片导入流程
```
用户拖拽 → Frontend → import_images(path)
  → scan_folder() 递归扫描
  → probe_dimensions() 提取尺寸
  → metadata::probe_metadata() 提取元数据
  → INSERT INTO images + images_fts
  → 返回 ImportResult
```

### 3.2 语义搜索流程
```
用户输入文本 → embed_text_cmd(text) → Ollama embedding
  → search_semantic_cmd(queryEmbedding)
  → sqlite-vec cosine search
  → 返回 SemanticSearchResult[]
  → 缓存结果到 LRU
```

### 3.3 AI 分析流程
```
用户点击分析 → analyze_image_cmd(imageId)
  → 读取图片文件 → base64 编码
  → 调用 Ollama llava API (120s timeout)
  → INSERT INTO analysis_history
  → apply_ai_tags_cmd(imageId) 自动标注
  → 返回 AnalysisResult
```

## 4. 关键设计决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 状态管理 | Zustand | 轻量，无 boilerplate，TypeScript 友好 |
| 数据库 | SQLite | 单文件，零配置，FTS5 原生支持 |
| 向量搜索 | sqlite-vec | 与 SQLite 集成，无需外部服务 |
| AI 引擎 | Ollama | 本地运行，隐私安全，模型可选 |
| 前端框架 | React 19 | 生态成熟，团队熟悉 |
| 构建工具 | Vite | 快速 HMR，原生 ESM |
