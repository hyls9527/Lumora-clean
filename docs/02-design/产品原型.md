### 2.2 工具栏

| 控件 | 类型 | 功能 | 代码依据 |
|------|------|------|----------|
| Creator Mode | Toggle | 切换显示 prompt/seed/model | `imageStore.setMode()` |
| Grid/List View | Toggle | 切换视图模式 | `imageStore.setView()` |
| Sort | Dropdown | 按时间/评分/模型/大小排序 | `imageStore.setSortBy()` |
| Model Filter | Dropdown | 按模型过滤 | `imageStore.setModelFilter()` |

### 2.3 ImageCard 交互

**悬停效果**：
- 阴影增强
- 微向上移动 (translateY -2px)

**键盘导航**：
- Tab 聚焦
- Enter 打开详情
- 方向键导航

**显示信息**：
- 图片预览
- 评分 (0-5 星)
- 收藏状态
- 标签列表
- 嵌入状态
- 相似度（搜索时）

**代码依据**：`src/components/ui/ImageCard.tsx`

### 2.4 无限滚动

**实现方式**：
- `LazyLoad` 组件包裹列表
- 滚动到底部自动加载下一页
- 每页 40 张图片

**代码依据**：
- 组件：`src/components/ui/LazyLoad.tsx`, `src/components/ui/InfiniteScroll.tsx`
