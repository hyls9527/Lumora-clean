# 产品原型 — 用户流程图

## 1. 核心用户流程

### 1.1 首次使用流程
```
启动应用
  │
  ├─ 侧边栏显示导航（Gallery/Search/Dashboard/Settings）
  ├─ 侧边栏底部显示 Ollama 状态（可选）
  │
  ▼
Gallery 页面（空状态）
  │
  ├─ 显示导入提示："拖拽文件夹到此处"
  │
  ▼
用户拖拽文件夹
  │
  ├─ 显示导入进度
  ├─ 完成后显示图片网格
  │
  ▼
浏览图片
```

### 1.2 图片浏览流程
```
Gallery 页面
  │
  ├─ 网格视图（默认）
  │   ├─ 图片缩略图
  │   ├─ 评分徽章（梅花印）
  │   ├─ 收藏标记（◆）
  │   └─ 标签徽章
  │
  ├─ 排序控制
  │   ├─ 按导入时间（默认）
  │   ├─ 按评分
  │   └─ 按收藏
  │
  ▼
点击图片
  │
  ├─ 打开 DetailModal
  │   ├─ 大图预览
  │   ├─ 元数据面板（prompt/seed/model/尺寸）
  │   ├─ 评分控件
  │   ├─ 收藏按钮
  │   ├─ 标签管理
  │   ├─ AI 分析结果
  │   └─ 变体组列表
  │
  ▼
操作图片
  │
  ├─ 评分 → update_rating
  ├─ 收藏 → toggle_favorite
  ├─ 添加标签 → add_tag_to_image
  ├─ AI 分析 → analyze_image_cmd
  └─ 删除 → soft_delete_image
```

### 1.3 搜索流程
```
Search 页面
  │
  ├─ 搜索栏
  │   ├─ 文本输入
  │   ├─ 模式切换：全文搜索 / 语义搜索
  │   └─ 高级设置（字段范围：prompt/seed/model）
  │
  ├─ 全文搜索路径
  │   ├─ 输入关键词 → search_images(query)
  │   └─ 返回 FTS5 匹配结果
  │
  ├─ 语义搜索路径
  │   ├─ 输入自然语言描述
  │   ├─ embed_text_cmd(text) → 向量
  │   ├─ search_semantic_cmd(embedding) → 结果
  │   └─ 显示相似度分数
  │
  └─ 图片搜索路径
      ├─ 选择参考图片
      ├─ searchByImage(path) → CLIP 嵌入
      └─ 显示视觉相似结果
```

### 1.4 设置流程
```
Settings 页面
  │
  ├─ 语言切换（中/英）
  │   └─ set_setting('language', 'zh'/'en')
  │
  ├─ 主题切换（明/暗）
  │   └─ set_setting('theme', 'light'/'dark')
  │
  ├─ 数据库备份
  │   ├─ 导出 → export_database(destination)
  │   └─ 导入 → import_database(source)
  │
  └─ Ollama 配置
      └─ 显示当前 OLLAMA_HOST 地址
```

## 2. 交互规范

### 2.1 命令面板 (⌘K)
- 输入框获得焦点
- 输入关键词过滤命令列表
- 方向键选择，Enter 执行
- Escape 关闭

### 2.2 批量操作
- Ctrl/⌘ + 点击多选图片
- 选中后顶部显示 BatchToolbar
- 操作：批量删除、批量添加标签、批量 AI 标注

### 2.3 拖拽导入
- 拖拽文件/文件夹到应用窗口
- 显示 DropOverlay 覆盖层
- 松手触发导入
- 显示导入进度和结果

## 3. 页面结构

| 页面 | 路径 | 说明 |
|------|------|------|
| Gallery | / | 图片网格/列表，主页面 |
| Search | /search | 全文 + 语义搜索 |
| Dashboard | /dashboard | 统计概览 |
| Settings | /settings | 语言/主题/备份 |
| Import | /import | 导入管理 |
| Export | /export | 导出管理 |
| Favorites | /favorites | 收藏列表 |
| Trash | /trash | 回收站 |
