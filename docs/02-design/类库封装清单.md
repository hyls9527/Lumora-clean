# 通用类库与前端库封装清单 — Lumora v0.5.1

> 基于 `src/lib/` 目录实际代码结构生成。

## 1. 通用类库（src/lib/）

### 1.1 tauri.ts — IPC 封装层
**职责：** 封装 Tauri invoke 调用，提供浏览器模式 mock 和错误处理。

| 导出 | 类型 | 说明 |
|------|------|------|
| `invoke<T>(cmd, args?)` | async function | Tauri IPC 调用封装，浏览器模式返回 mock 数据 |
| `onWriteCommand(cb)` | function | 注册写命令完成回调（用于缓存失效） |
| `isTauriAvailable` | boolean | 是否在 Tauri 环境中运行 |
| `convertFileSrc(filePath)` | async function | 文件路径转 asset URL |

**特性：**
- 写命令完成后自动触发已注册的回调
- invoke 失败时 console.error 记录但不吞异常
- 错误消息通过 i18n 映射为用户友好文本

### 1.2 tokens.ts — 设计令牌
**职责：** 将 DESIGN.md 设计语言转为可编程的 TypeScript 常量。

| 分类 | 示例 |
|------|------|
| Colors | `bg`, `bgAlt`, `surface`, `border`, `text`, `accent`, `danger`, `success` |
| Shadows | `card`, `cardHover`, `elevated` |
| Typography | `fontDisplay`, `fontBody`, `fontMono` |
| Spacing | `space1` ~ `space9` (4/8/12/16/20/24/32/48/64px) |
| Radius | `radiusCard`(2px), `radiusButton`(4px), `radiusInput`(4px), `radiusPopup`(6px) |
| Transition | `transition` (200ms ease-out) |

### 1.3 format.ts — 格式化工具
**职责：** 提供日期、文件大小等格式化函数。

| 导出 | 说明 |
|------|------|
| `formatDate(iso)` | ISO 时间戳 → zh-CN 本地化字符串 |
| `formatFileSize(bytes)` | 字节数 → 可读文件大小 |

### 1.4 i18n.ts — 国际化
**职责：** 提供中英文翻译切换。

| 导出 | 说明 |
|------|------|
| `useTranslation()` | Hook，返回 `t(key)` 翻译函数 |
| 语言资源 | `src/i18n/zh.json`、`src/i18n/en.json` |

## 2. 业务 API 层（src/lib/api/）

### 2.1 images.ts — 图片 API
**职责：** 封装图片相关的 Tauri 命令调用。

| 函数 | 对应 Tauri 命令 |
|------|----------------|
| `importImages(path)` | `import_images` |
| `listImages(page, perPage)` | `list_images` |
| `searchImages(query)` | `search_images` |
| `searchImagesAdvanced(query, field?)` | `search_images_advanced` |
| `updateRating(id, rating)` | `update_rating` |
| `toggleFavorite(id)` | `toggle_favorite` |
| `listFavorites()` | `list_favorites` |
| `batchSoftDelete(ids)` | `batch_soft_delete` |
| `getVariantGroupImages(id)` | `get_variant_group_images` |
| `exportImages(ids, destDir, format, renameTemplate?)` | `export_images` |

### 2.2 ai.ts — AI 分析 API
| 函数 | 说明 |
|------|------|
| `analyzeImage(imageId, imagePath?)` | 调用 AI 分析图片 |
| `getAnalysisResult(imageId)` | 获取最新分析结果 |
| `getAnalysisHistory(imageId)` | 获取分析历史 |
| `batchAutoTag(imageIds)` | 批量 AI 自动标注 |

### 2.3 embeddings.ts — 嵌入 API
| 函数 | 说明 |
|------|------|
| `getEmbeddingStatus(imageId)` | 获取嵌入状态 |
| `getEmbeddingStats()` | 获取嵌入统计 |
| `generateEmbedding(imageId, embedding)` | 存储嵌入向量 |
| `embedText(text, model?)` | 文本转向量 |
| `generateEmbeddingForImage(imageId, desc, model?)` | 为图片生成嵌入 |

### 2.4 semantic.ts — 语义搜索 API
| 函数 | 说明 |
|------|------|
| `searchSemantic(queryEmbedding, limit?)` | 向量相似度搜索 |
| `searchByImage(imagePath)` | 图片到图片搜索（CLIP） |

### 2.5 semanticCache.ts — 语义搜索缓存
**职责：** Map-based LRU 缓存，减少重复搜索调用。

| 特性 | 值 |
|------|---|
| 实现 | Map + 按访问顺序排列 |
| TTL | 30 分钟 |
| 最大条目 | 100 |
| 最大大小 | 5MB |
| 驱逐策略 | LRU |
| 持久化 | localStorage（延迟写入，防抖） |

### 2.6 backup.ts — 数据库备份 API
| 函数 | 说明 |
|------|------|
| `exportDatabase(destination)` | 导出 SQLite 文件 |
| `importDatabase(source)` | 导入 SQLite 文件 |
