## Manual Steps Required

1. **Python sidecar build** — Requires Python 3.13 + PyTorch 2.12.1 CPU:
   ```
   cd src-tauri/sidecar
   pip install -r requirements.txt
   python build.py
   ```

2. **MSI production build** — After sidecar binary exists:
   ```
   npm ci && npm run build && cargo tauri build --bundles msi
   ```
