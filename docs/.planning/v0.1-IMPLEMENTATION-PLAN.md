# Lumora v0.1 实施计划

> 最小可用：图库 + 导入 + 搜索

## 前置条件

- [x] Node.js v24.18.0
- [x] Rust 1.96.0 + Cargo 1.96.0
- [x] Tauri CLI 2.11.3
- [x] MSVC Build Tools 2022
- [x] HTML 原型 11 页（pages/）
- [x] 设计规范（docs/DESIGN.md）
- [x] 架构文档（docs/ARCHITECTURE.md）

## 阶段 1：骨架搭建（串行）

**目标：** Tauri + React 19 + Tailwind v4 + TypeScript 项目可编译运行

| 步骤 | 工作 | 验证 |
|------|------|------|
| 1.1 | `cargo tauri init` 初始化 Tauri 项目 | `src-tauri/` 目录存在，`Cargo.toml` 可编译 |
| 1.2 | 初始化 React 19 + Vite + TypeScript 前端 | `npm run dev` 可启动 |
| 1.3 | 配置 Tailwind CSS v4 + DESIGN.md 设计令牌 | 页面渲染暖白色背景 |
| 1.4 | 创建目录结构（src/components, src/features, src/lib, src/stores） | 目录存在 |
| 1.5 | 配置 TypeScript strict + ESLint | `npx tsc --noEmit` 零错误 |
| 1.6 | 验证 Tauri dev 模式可运行 | `cargo tauri dev` 窗口弹出 |

**产出：** 可编译运行的空壳应用

## 阶段 2：UI 移植 + 数据层（并行）

### 2A：UI 移植（子代理 A）

**目标：** HTML 原型转为 React 组件

| 步骤 | 工作 | 来源 | 验证 |
|------|------|------|------|
| 2A.1 | 侧边栏组件（Sidebar） | pages/gallery.html | 导航可点击 |
| 2A.2 | 图库页面（GalleryPage） | pages/gallery.html | 图片网格渲染 |
| 2A.3 | 导入页面（ImportPage） | pages/import.html | 文件选择器可打开 |
| 2A.4 | 搜索栏组件（SearchBar） | pages/search.html | 输入框可输入 |
| 2A.5 | 图片卡片组件（ImageCard） | pages/gallery.html | 卡片显示图片+元数据 |
| 2A.6 | 详情面板（DetailPanel） | pages/artwork-detail.html | 点击图片弹出详情 |

**规则：**
- 遵循 DESIGN.md（无图标、暖白色、DM Sans、200ms 过渡）
- 使用 mock 数据（Zustand store + 静态 JSON）
- 所有文本中英文 i18n

### 2B：数据层（子代理 B）

**目标：** SQLite + Tauri commands 可用

| 步骤 | 工作 | 验证 |
|------|------|------|
| 2B.1 | SQLite schema v1（images 表） | `cargo test` 通过 |
| 2B.2 | Tauri command: import_images（文件夹扫描 + 元数据提取） | IPC 调用返回结果 |
| 2B.3 | Tauri command: list_images（分页查询） | IPC 调用返回图片列表 |
| 2B.4 | Tauri command: search_images（FTS5 全文搜索） | IPC 调用返回搜索结果 |
| 2B.5 | 缩略图生成（512x512 webp） | 缩略图文件存在 |
| 2B.6 | Schema 迁移机制 | 版本号可递增 |

**规则：**
- Clean Architecture：Domain 层零外部依赖
- Commands 层 ≤30 行/函数
- 所有写操作在事务内

## 阶段 3：功能集成（串行）

**目标：** UI 真实调用后端，不再用 mock 数据

| 步骤 | 工作 | 验证 |
|------|------|------|
| 3.1 | 替换 mock API → Tauri invoke() | 图库显示真实导入图片 |
| 3.2 | 导入流程：文件选择器 → 扫描 → 入库 → 显示 | 完整导入链路 |
| 3.3 | 搜索流程：输入 → FTS5 → 结果渲染 | 搜索延迟 ≤100ms |
| 3.4 | 键盘导航（方向键、Enter） | 图库可键盘操作 |
| 3.5 | Loading / Error 状态 | 骨架屏 + 重试按钮 |

## 风险

| 风险 | 缓解 |
|------|------|
| Tauri init 失败 | 检查 MSVC 版本，手动创建 Cargo.toml |
| Tailwind v4 配置复杂 | 参考 DESIGN.md 设计令牌直接写 CSS 变量 |
| HTML 原型结构不标准 | 手动提取关键样式和布局，不盲目复制 |
| SQLite FTS5 中文分词 | v0.1 先用英文搜索，中文分词后续阶段 |
