# v0.1 MVP Frontend — Requirements

## Design System (DESIGN-01)
- DESIGN-01-01: All colors use CSS variables from @theme, zero hardcoded hex
- DESIGN-01-02: 4-layer Notion shadows (not 2-layer)
- DESIGN-01-03: 100ms Linear-style transitions (not 150ms)
- DESIGN-01-04: DM Sans font, zero Inter/Roboto
- DESIGN-01-05: Pure black/white banned (#1c1917 / #faf9f7)

## Component Quality (COMP-01)
- COMP-01-01: All interactive elements have hover + focus-visible states
- COMP-01-02: Cards use 12px radius, buttons 9999px (pill)
- COMP-01-03: Card padding ≥ 24px
- COMP-01-04: Image grid gap ≥ 12px
- COMP-01-05: Labels de-emphasized (10-11px, uppercase, muted)

## Feature Completeness (FEAT-01)
- FEAT-01-01: Command palette (⌘K) with search
- FEAT-01-02: Keyboard shortcuts (arrows navigate, space select, delete trash)
- FEAT-01-03: Responsive layout (sidebar collapses on narrow)
- FEAT-01-04: Empty states for all pages
- FEAT-01-05: Loading skeletons

## Code Quality (CODE-01)
- CODE-01-01: `npx tsc --noEmit` zero errors
- CODE-01-02: `vite build` clean
- CODE-01-03: No `any` types
- CODE-01-04: Components ≤ 200 lines
