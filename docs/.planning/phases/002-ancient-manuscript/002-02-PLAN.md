# Plan 002-02: Sidebar + Layout

## Task 1: Rewrite Sidebar.tsx

<read_first>
- src/components/Sidebar.tsx
</read_first>

<action>
Rewrite the sidebar to match the 书脊 (book spine) design. Use cn() for conditional classes. Use useTranslation() for all text.

1. Width: w-[200px] (narrow book spine)
2. Background: bg-surface
3. Right border: border-r border-border
4. Gold line decoration: use a div with absolute positioning on the right edge:
   - className="absolute top-[60px] bottom-[60px] right-0 w-px"
   - style={{ background: 'linear-gradient(to bottom, transparent, rgba(122,92,18,0.12) 20%, rgba(122,92,18,0.12) 80%, transparent)' }}
5. Logo area (px-5 pt-9 pb-7 border-b border-border-subtle):
   - Logo text: className="font-serif text-[20px] font-light tracking-[0.08em] text-text" — content "Lumora"
   - Subtitle: className="text-[9px] uppercase tracking-[0.2em] text-text-muted mt-2 font-sans font-normal" — use t("sidebar.subtitle") key
6. Search (mx-4 my-5 px-3 py-2 rounded-md bg-bg border border-border-subtle):
   - Icon: opacity-0.4, text-[11px]
   - Text: font-serif text-[12px] text-text-muted font-light
   - ⌘K hint: hidden by default (opacity-0 group-hover:opacity-50), font-serif text-[9px] text-text-faint, no border
   - Wrap button in className="group" to enable hover on hint
7. Nav (flex-1 px-3):
   - Items: font-serif text-[13px] font-normal text-text-secondary
   - Active: font-semibold text-text, with ::before pseudo (left-0.5, w-0.5, h-4, bg-accent, rounded-full)
   - Active spacing: mt-1.5 mb-2 (breathing room)
   - Dot: w-[3px] h-[3px] rounded-full bg-text-faint, active: bg-accent
8. Stats (mx-4 mb-6 px-4 py-3.5 bg-bg rounded-md border border-border-subtle):
   - Rows: border-b border-dotted with rgba(139,115,75,0.06)
   - Labels: text-[9px] uppercase tracking-[0.1em] text-text-muted font-sans font-normal
   - Values: font-sans text-[13px] text-text tabular-nums font-medium
   - Star icon for favorites: use text-accent (gold) not fill
</action>

<acceptance_criteria>
- Sidebar width is 200px
- Logo is "Lumora" without any symbol, serif font, weight 300
- Nav items use serif font
- Active nav has gold left bar (2px wide, 16px tall)
- Stats use dotted dividers (border-dotted, not border-solid)
- Gold line decoration on right edge (gradient pseudo-element)
- ⌘K hint is hidden by default, appears on hover
- Active nav has breathing margin (mt-1.5 mb-2)
</acceptance_criteria>
