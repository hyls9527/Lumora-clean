# Plan 004-03: Tag Management

## Task 1: Create TagManager component

<action>

4. Tag filtering in gallery view
5. Tag cloud visualization

Design rules (古卷·灯火):
- Tag pills: rounded-[4px], font-serif, bg-accent-subtle

- Color picker: muted accent colors only
