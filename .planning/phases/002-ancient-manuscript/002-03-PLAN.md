# Plan 002-03: Gallery + ImageCard

## Task 1: Rewrite ImageCard.tsx

<read_first>
- src/components/ImageCard.tsx
</read_first>

<action>
Rewrite ImageCard to match the 书页 (book page) design:

1. Card container: rounded-[6px] (not 12px), shadow-card, hover shadow-card-hover
2. NO hover scale (too playful) — only shadow deepens
3. Transition: 200ms ease-out
4. Rating: 梅花印 (plum flower stamp) — not stars, not dots
   - SVG component: 5 petals + 1 pistil
   - filled: accent color fill, 0 0 4px rgba(139,105,20,0.25) glow
   - empty: stroke-only outline, rgba(139,115,75,0.15), no fill
   - size: 14x14px viewBox
   - position: absolute bottom-2 left-2, gap-1
   - Create src/components/ui/PlumRating.tsx as a reusable SVG component
   ```tsx
   function PlumFlower({ filled }: { filled: boolean }) {
     return (
       <svg width="14" height="14" viewBox="0 0 14 14">
         {[0, 72, 144, 216, 288].map(angle => (
           <ellipse key={angle} cx="7" cy="3" rx="2.5" ry="3.5"
             transform={`rotate(${angle} 7 7)`}
             fill={filled ? 'var(--color-accent)' : 'none'}
             stroke={filled ? 'none' : 'rgba(139,115,75,0.2)'}
             strokeWidth="0.8"
           />
         ))}
         <circle cx="7" cy="7" r="1.5"
           fill={filled ? 'var(--color-accent)' : 'rgba(139,115,75,0.15)'}
         />
       </svg>
     );
   }
   ```
5. Favorite: stamp not heart
   - 20px square, rounded-[3px], bg-accent/80, text "◆"
   - color: #f2ede4
   - position: absolute top-2 right-2
   - opacity 0 by default, 1 on hover or when favorited
6. Format badge: sans-serif 8px uppercase, bg-black/25, opacity 0 on hover
7. Selection: ring-2 ring-accent ring-offset-1 (not ring-offset-2)
8. Image: no scale on hover, just object-cover
</action>

<acceptance_criteria>
- Card radius is 6px
- No image hover scale
- Rating uses 梅花印 plum flower SVG (✿) — filled=accent+glow, empty=stroke-only outline
- Favorite uses stamp (◆) not heart
- Transitions are 200ms ease-out
- Shadow deepens on hover (warm brown tones)
</acceptance_criteria>

2. Masonry grid: CSS columns (4 columns), column-gap 16px
