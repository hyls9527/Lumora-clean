# Plan 002-02: Sidebar + Layout

## Task 1: Rewrite Sidebar.tsx

<read_first>
- src/components/Sidebar.tsx
</read_first>

<action>
Rewrite the sidebar to match the 书脊 (book spine) design:

1. Width: 200px (narrow, like a book spine)
2. Background: bg-surface (#f7f2ea)
3. Right border: 1px solid border
4. Gold line decoration: pseudo-element on right edge, gradient from transparent to accent (opacity 0.12) to transparent, positioned top:60px bottom:60px
5. Logo: "Lumora" in Noto Serif SC, weight 300, letter-spacing 0.08em (no star/symbol)
6. Subtitle: "光之韵律 · 图片典藏" in DM Sans, 9px, uppercase, letter-spacing 0.2em, text-muted
7. Search: bg-bg, border 1px subtle, search icon opacity 0.4, hint text in serif font no border
8. Nav items: serif font, dot indicator (3px circle), active has left 2px gold bar
9. Active nav: font-weight 600, margin-top 6px margin-bottom 8px (breathing)
10. Stats: bg-bg, border 1px subtle, dotted dividers, labels 9px uppercase, values DM Sans tabular-nums weight 500

Use cn() for conditional classes. Use useTranslation() for all text.
</action>

<acceptance_criteria>
- Sidebar width is 200px
- Logo is "Lumora" without any symbol
- Nav items use serif font
- Active nav has gold left bar (2px wide, 16px tall)
- Stats use dotted dividers, not solid

</acceptance_criteria>
