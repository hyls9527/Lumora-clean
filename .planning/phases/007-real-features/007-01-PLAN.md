## Task 2: Add open_folder_dialog command

<action>
Add a command that opens a native folder picker and imports images:

```rust
#[tauri::command]
async fn open_folder_dialog(state: State<'_, AppState>) -> Result<import::ImportResult, String> {
    use tauri_plugin_dialog::DialogExt;
    
    let folder = state.app_handle.dialog()
        .blocking_pick_folder()
        .ok_or("No folder selected")?;
    
    let folder_path = folder.as_path()
        .ok_or("Invalid folder path")?
        .to_path_buf();
    
    let db = state.db.lock().map_err(|e| e.to_string())?;
    import::import_folder(&folder_path, &db)
}
```

Note: This requires storing the app handle in AppState.
</action>

<acceptance_criteria>
- Native folder picker opens
- Selected folder is imported
- Returns ImportResult with counts
</acceptance_criteria>
