## Task 2: Update search to use FTS5

<action>
Update search_images in db.rs to use FTS5:
```rust
pub fn search_images(&self, query: &str) -> Result<Vec<ImageRecord>> {
    let mut stmt = self.conn.prepare(
        "SELECT i.id, i.file_path, i.file_hash, i.file_size_kb, i.width, i.height, i.format, 
                i.created_at, i.rating, i.favorite, i.metadata_json
         FROM images i
         JOIN images_fts fts ON i.id = fts.rowid
         WHERE images_fts MATCH ?1 AND i.deleted = 0
         ORDER BY rank
         LIMIT 100"
    )?;
    
    let images = stmt.query_map([query], |row| {
        Ok(ImageRecord { ... })
    })?.collect::<Result<Vec<_>>>()?;
    
    Ok(images)
}
```
</action>

<acceptance_criteria>
- Search uses FTS5 MATCH syntax
- Results are ranked by relevance
- Fallback to LIKE if FTS5 fails
</acceptance_criteria>
