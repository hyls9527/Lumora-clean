Create `src/stores/ai-analysis-store.ts` following the exact Zustand `create<State>` pattern from `src/stores/embedding-store.ts` — interface first, then `create<State>((set, get) => ({...}))`. Import types from `@/lib/api/analysis` and functions `analyzeImage`, `getAnalysisResult`, `getAnalysisHistory`.

**Define `AiAnalysisState` interface:**

```typescript
interface AiAnalysisState {
  results: Map<string, AnalysisResult | null>
  history: Map<string, AnalysisHistoryEntry[]>
  states: Map<string, 'idle' | 'analyzing' | 'complete' | 'error'>
  progress: Map<string, number>
  errors: Map<string, string | null>
  acceptedTags: Map<string, Set<string>>
  rejectedTags: Map<string, Set<string>>

  // Getters
  getResult: (imageId: string) => AnalysisResult | null
  getHistory: (imageId: string) => AnalysisHistoryEntry[]
  getState: (imageId: string) => 'idle' | 'analyzing' | 'complete' | 'error'
  getProgress: (imageId: string) => number
  getError: (imageId: string) => string | null
  isTagAccepted: (imageId: string, tag: string) => boolean
  isTagRejected: (imageId: string, tag: string) => boolean

  // Actions
  triggerAnalysis: (imageId: string) => Promise<void>
  acceptTag: (imageId: string, tag: string) => void
  rejectTag: (imageId: string, tag: string) => void
}
```

**Initial state:** All Maps empty — `new Map()`.

**Getter implementations:**
- `getResult(imageId)`: Return `state.results.get(imageId) ?? null`.
- `getHistory(imageId)`: Return `state.history.get(imageId) ?? []`.
- `getState(imageId)`: Return `state.states.get(imageId) ?? 'idle'`.
- `getProgress(imageId)`: Return `state.progress.get(imageId) ?? 0`.
- `getError(imageId)`: Return `state.errors.get(imageId) ?? null`.
- `isTagAccepted(imageId, tag)`: Return `state.acceptedTags.get(imageId)?.has(tag) ?? false`.
- `isTagRejected(imageId, tag)`: Return `state.rejectedTags.get(imageId)?.has(tag) ?? false`.

**`triggerAnalysis(imageId)` action:**
1. Set state to 'analyzing', progress to 0, clear error: update `states`, `progress`, `errors` Maps for this imageId.
2. Start a `setInterval` at 200ms that increments progress by `Math.floor(Math.random() * 8) + 3` (3-10 per tick). When progress >= 100, clear the interval and set progress to 100. Store the interval ref in a module-level variable `let progressIntervals = new Map<string, ReturnType<typeof setInterval>>()` — keyed by imageId so multiple images don't interfere.
3. Wrap the API call in try/catch. Call `await analyzeImage(imageId)`.
4. On success: Update `results` Map with the result. Set state to 'complete'. Set progress to 100. Clear interval if still running.
5. Also call `await getAnalysisHistory(imageId)` and store in `history` Map.
6. On error (catch): Set state to 'error'. Set `errors` Map with `String(err)`. Clear interval.

**`acceptTag(imageId, tag)` action:**
Use functional `set()` to create a new Set from the current acceptedTags for this imageId, add the tag, and update. Ensure the imageId key exists in the Map after this (create new entry if absent).

**`rejectTag(imageId, tag)` action:**
Same pattern as acceptTag — create new Set, add tag, update `rejectedTags` Map.

**Module-level cleanup variable:** `const progressIntervals = new Map<string, ReturnType<typeof setInterval>>()` — cleared when triggerAnalysis completes or errors for a given imageId.

Export as: `export const useAiAnalysisStore = create<AiAnalysisState>((set, get) => ({...}))`

Use `console.error()` in catch blocks matching existing store pattern. All state updates use functional `set()` returning partial state.
