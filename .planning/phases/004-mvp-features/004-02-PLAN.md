# Plan 004-02: Keyboard Shortcuts

## Task 1: Add keyboard navigation to GalleryPage

<action>
Add keyboard event handlers to GalleryPage for:

1. **Arrow keys**: Navigate between images in the grid
   - Left/Right: Move horizontally
   - Up/Down: Move vertically
2. **Space**: Toggle selection on focused image
3. **Enter**: Open detail panel for focused image
4. **Delete/Backspace**: Delete focused image (move to trash)
5. **Escape**: Clear selection or close detail panel
6. **⌘K / Ctrl+K**: Open command palette (already exists)

Implementation:
- Track focused index in app-store
- Add tabIndex to image cards for focus management
- Add visual focus indicator (gold ring) on focused card
- Prevent default browser behavior for these keys
</action>

<acceptance_criteria>
- Arrow keys navigate between images
- Space toggles selection
- Enter opens detail panel
- Delete moves to trash
- Focus ring visible on current card
- Works with both masonry and virtualized layouts
</acceptance_criteria>
